# CSV の任意の列の抽出

```
usage: csvselect [-h] [-s] [-t] [-n NUMBER] [-e REGEXP [REGEXP ...]] file

positional arguments:
  file                  file name

optional arguments:
  -h, --help            show this help message and exit
  -s                    set encoding to SJIS-DOS
  -t                    set separator to \\t (default ,)
  -n NUMBER             set NUMBER
  -e REGEXP [REGEXP ...]
                        set regexp PATTERN
```
